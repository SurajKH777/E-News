# 📰 E-News — Daily Times

A simple, responsive newspaper-style news web portal built with HTML & CSS.

## 🌐 Live Demo

👉 [View Live on GitHub Pages](https://surajkh777.github.io/E-News/)

## 📂 Project Structure

```
E-News/
├── index.html      # Main news portal page
└── README.md       # Project documentation
```

## 🚀 Features

- Classic newspaper layout (3-column: Left, Main, Right)
- Responsive design — works on mobile & desktop
- Breaking news, Weather, IPL 2026, Business sections
- Clean typography using Georgia serif font

## 🛠️ Tech Stack

- HTML5
- CSS3 (Vanilla)
